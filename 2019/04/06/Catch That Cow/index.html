Catch That Cow (bfs)

Posted on 2019-04-06
  • Time limit: 2000 ms
  • Memory limit: 65536 kB

HDU - 2717 ; POJ - 3278

题干：

Description:

Farmer John has been informed of the location of a fugitive cow and wants to catch her immediately. He starts at a point N (0 ≤ N ≤ 100,000) on a number line and the cow is at a point K (0 ≤ K ≤ 100,000) on the same number line. Farmer John has two modes of transportation: walking and teleporting.

  • Walking: FJ can move from any point X to the points X - 1 or X + 1 in a single minute
  • Teleporting: FJ can move from any point X to the point 2 × X in a single minute.

If the cow, unaware of its pursuit, does not move at all, how long does it take for Farmer John to retrieve it?

  • 2019/3/31

Input

Line 1: Two space-separated integers: N and K

Output

Line 1: The least amount of time, in minutes, it takes for Farmer John to catch the fugitive cow.

Sample Input

5 11

Sample Output:

4

题解

  • 广度优先搜索的经典题目
  • 我们使用一个队列来维护当前已经走到的所有的点，并且
    用一个数组time[]记录使用的时间。我们每次从队列中取出一个点，
    然后执行之前的两种操作（N+=1，N*=2），若得到的值j未曾加入到队列中，就
    将j放到队列里，并令time[j] = time[i] + 1。当第一次j == K时，此
    时的distj便为答案。
    因为我们使用的是广搜，所以说明当我们找到了鞆要走i步才能
    到达的点时，所有小于i步就能到达的点已经被找过了。所以，
    我们每次找到一个新的点，这个点此时的time一定为可能的最小值。

Source Code

#include <iostream>
#include <cstring>
#include <queue>

using namespace std;

queue<int> q;
int N, K;
int time[100010];
int visited[100010];//标记是否走过


int solve(){
    while (!q.empty() ){
        int now = q.front();
        q.pop();
        int next;
        for (int i = 0; i < 3; i++){
            if (i == 0) next = now - 1;
            if (i == 1) next = now + 1;
            if (i == 2) next = now * 2;
            if (next < 0 || next > 100010) continue;//注意不要越界
            if (!visited[next]){
                q.push(next);
                visited[next] = 1;
                time[next] = time[now] + 1;
            }
            if (next == K) return time[next];
        }
    }
    return 0;
}


int main(){
    memset(time, 0, sizeof(time) );
    memset(visited, 0, sizeof(visited) );

    cin >> N >> K;
    if (N >= K){
        cout << N - K << endl;
    } else {
        visited[N] = 1;
        time[N] = 0;
        q.push(N);
        cout << solve() << endl;
    }


    return 0;
}
